# 🤖 بوت الصف الثالث الثانوي

بوت تليجرام كامل لطلاب الثانوية العامة: كل الكتب وكتب المدرسين لكل الشعب،
بالإضافة لمعسكرات تجميع المنهج (شهر / 20 يوم / 15 يوم) متاحة بس بعد دعوة 7 أصدقاء.
شغال بالكامل على GitHub Actions زي بوت القرآن، من غير أي سيرفر خاص.

---

## 1) إنشاء البوت من BotFather
1. افتح [@BotFather](https://t.me/BotFather) وابعتله `/newbot`.
2. اختار اسم ويوزر للبوت.
3. هيديك **التوكن** - احفظه، هتحتاجه في الخطوة الجاية.

## 2) إضافة التوكن كـ Secret في المستودع
1. روح لمستودعك على GitHub > **Settings** > **Secrets and variables** > **Actions**.
2. اضغط **New repository secret**.
3. الاسم: `BOT_TOKEN`
4. القيمة: التوكن اللي خدته من BotFather.
5. احفظ.

> ملحوظة: متحطش التوكن في أي ملف كود، خليه دايمًا في الـ Secrets بس.

## 3) رفع الكتب
كل مادة ليها مجلد جاهز جوه `books/`. روح للمجلد المناسب وارفع فيه ملفين بالاسم بالظبط:
- `student.pdf` (كتاب الطالب)
- `teacher.pdf` (كتاب المدرس)

مثال: كتاب الفيزياء لشعبة علمي علوم بيتحط في:
`books/science_science/physics/student.pdf`

نفس الكلام لمعسكرات تجميع المنهج، بترفعهم في `books/camps/` بالأسماء:
`month_plan.pdf` / `20days_plan.pdf` / `15days_plan.pdf`

كل مجلد فيه ملف نصي بيوضحلك بالظبط اسم الملف المطلوب.

> هتلاقي فيه حد أقصى لحجم الملف اللي البوت يقدر يبعته عبر Telegram وهو 50 ميجا تقريبًا.
> لو كتاب أكبر من كده، قسّمه لأجزاء أو ارفعه بجودة أقل.

## 4) تعديل أو إضافة مواد جديدة
كل حاجة بتتحكم فيها من ملف **`content.json`** بدون ما تلمس الكود خالص:
- تقدر تغيّر أسماء المواد، تضيف مادة جديدة، أو تضيف شعبة كاملة.
- لو عايز تفعّل الاشتراك الإجباري في قناة قبل استخدام البوت (زي بوت القرآن بالظبط)،
  غيّر في آخر الملف:
  ```json
  "requireSubscribe": true,
  "channelUsername": "@اسم_قناتك"
  ```
- لو عايز تغيّر عدد الدعوات المطلوبة لفتح المعسكرات (افتراضيًا 7):
  ```json
  "requiredInvites": 7
  ```

## 5) تشغيل البوت
1. روح لتبويب **Actions** في المستودع.
2. اختار workflow اسمه **"تشغيل بوت الثانوية العامة"**.
3. اضغط **Run workflow** لأول مرة يدويًا.
4. من بعد كده هيشتغل لوحده تلقائيًا ويعيد تشغيل نفسه قبل ما يوصل لحد الـ6 ساعات بتاعة GitHub، فهيفضل شغال 24 ساعة من غير تدخل منك.

---

## ⚙️ ازاي نظام الدعوات شغال
- كل مستخدم بياخد رابط دعوة خاص بيه: `https://t.me/اسم_البوت?start=ref_<ID بتاعه>`
- لما حد جديد يدخل من الرابط ده، صاحب الرابط بياخد +1 في عداد دعواته.
- لما يوصل لـ7 دعوات، أزرار المعسكرات بتتفتح تلقائيًا قدامه.
- البيانات بتتسجل في `data/users.json` وبتتحفظ تلقائيًا في المستودع بعد كل تشغيل.

## 🎨 شكل الأزرار
الأزرار متظبطة تبقى زرار واحد عريض في كل صف بنفس ترتيب بوت القرآن
(اشترك الآن / تحقق من الإشتراك). اللون الأخضر اللي شايفه في تطبيق تليجرام
بييجي من ثيم تطبيقك مش من الكود، يعني هيفضل بنفس شكل بوتاتك التانية تلقائيًا.

## ✍️ توقيع البوت
كل رسالة وكل ملف بيتبعت من البوت بيتحط تحته تلقائيًا سطر باسم اليوزر بتاع البوت
(`🤖 @اسم_البوت`) زي ما طلبت بالظبط.

---

### بنية المشروع
```
bot.js                 -> كود البوت كامل
content.json           -> كل المواد والشعب والمعسكرات (تقدر تعدله بسهولة)
data/users.json        -> بيانات المستخدمين والدعوات (بتتحدث تلقائيًا)
books/                 -> هنا بترفع كل ملفات PDF
.github/workflows/bot.yml -> ملف تشغيل البوت على GitHub Actions
```
